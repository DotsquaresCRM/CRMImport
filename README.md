# MS Sql To CRM Import

This is XRMToolBox Plugin to insert MYSQL record in Dynamic CRM 365. This is easyest way to import data.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

First, Download XRM tool box from here: https://github.com/MscrmTools/XrmToolBox/releases/download/v.1.2018.7.28/XrmToolbox.zip.


### Uses

A step by step series of examples that tell you how to use this tool.

1. Download plugin(MS Sql To CRM Import) from xrm plugin store.
XRMToolBox -> Tools -> Plugin Store.

2. Search in Plugins Tab (MS Sql To CRM Import) of XRMToolBox.

3. Click on Plugin and Connect with you dynamic CRM Instance.

4. Now, Click on Load MS Sql Tables button and Connect with MS SQL server with Connection String or Database Credentails.

5. Select Tables that you want to import into CRM.

6. You get selected table column in List.

7. Same selection process doing at select Entities end.

8. Now, select column mapped with Entites Fiels.

9. You can up and down posiion of table and Entites column.

10. Insert Bulk record Import size(Max. 1000).

11. Click on Start Import.Now, Import Processing started.
